# Intégration maquette simple

![Helium final](./Helium_final.png)

Visitez la page : https://projetsdevfanny.github.io/TP1-2-3_FromScratch_Helium_JS_CSS/

Ce repo contient une intégration d'une maquette simple en HTML/CSS, avec deux variantes de mise en page et un petit effet JS.

## Contenu
- Version Grid (active dans `style.css`)
- Version Flexbox (commentée dans `style.css`)
- Responsive : iPhone et iPad
- Effet JS : `mousemove`

## Responsive (liens)
- [iPad](./Responsive_Ipad.png)
- [Mobile](./Responsive_mobile.png)
- [Mobile (suite)](./Responsive_mobile_suite.png)

## Où sont les 2 versions ?
Les deux variantes sont dans `style.css` :
- Les règles Grid sont actives par défaut
- Les règles Flexbox sont commentées (décommenter les blocs `display: flex` si vous voulez tester)

## Notes
- Les mises en page sont optimisées pour un écran 22"

## Lancer le projet
Ouvrez simplement le fichier `index.html` dans votre navigateur.
